## Mô tả

- Các phần màu (không phải màu trắng hoặc trong suốt) sẽ chuyển thành màu đen
- Các phần màu trắng và trong suốt sẽ giữ nguyên
- Kết quả được lưu dưới dạng PNG trong thư mục "output"
